# ImageInAction (Legacy)

ImageInAction is a simple application for creating drawings and animations on PC and Android devices.
## Features and tools:
 - pen;
 - eraser;
 - bucket;
 - rectangle selection and lasso selection;
 - undo/redo;
 - copy/paste of frame or selection;
 - zoom;
 - rotation;
 - possibility to play the animation with different frame rates;
 - shadow frame to see a selected frame behind the current one (see video).

**WARNING:** This is only a prototype, you may encounter bugs.
